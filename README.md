# Notepad++ Swift

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

A pure-Swift port of [Notepad++](https://notepad-plus-plus.org) to macOS — PowerEditor (the application), Scintilla (the editor engine), and Lexilla (the lexers) all rewritten in Swift. The goal is a native Mac app that preserves Notepad++'s functional spirit and muscle memory while feeling like a real Mac citizen.

## Status

🚧 Early development. No public release yet. The repo is public from day 1, but there's no shipping app to download — Phase 1 is still scaffolding the build/sign/notarize pipeline. Star/watch for updates.

## Requirements

macOS 14 (Sonoma) or later. Universal Binary planned (Apple Silicon + Intel).
